# Планировщик задач

Минимальный онлайн-планировщик задач (todo list). Vanilla HTML/CSS/JS, без сборки, без зависимостей. Данные хранятся в `localStorage` браузера.

## Запуск

В корне проекта:

```bash
python3 -m http.server 8000
```

Затем открыть [http://localhost:8000](http://localhost:8000) в браузере.

Альтернатива на Node:

```bash
npx --yes http-server -p 8000 .
```

## Возможности

- Добавление задачи (Enter или кнопка «Добавить»)
- Отметка выполнения / снятие отметки (чекбокс)
- Удаление одной задачи (×)
- Удаление всех выполненных задач (кнопка появляется автоматически)
- Счётчик оставшихся задач
- Сохранение между перезагрузками страницы (localStorage)

## Структура

```
index.html   разметка
style.css    стили
app.js       логика (модуль IIFE, без зависимостей)
```

## Ручная проверка

1. Открыть страницу → должен показаться пустой список с текстом «Задач пока нет.»
2. Ввести «купить молоко», нажать Enter → задача появилась, счётчик «Осталось: 1 из 1».
3. Поставить чекбокс → задача зачёркнута, появилась кнопка «Удалить выполненные».
4. Перезагрузить страницу → состояние сохранилось.
5. Нажать × у задачи → задача удалена.
6. Нажать «Удалить выполненные» → все выполненные задачи удалены.
